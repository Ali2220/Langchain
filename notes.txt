1. LangChain Ke 6 Core Concepts:

Models -->	AI ka dimaag (Gemini, GPT)	Jaise insaan ka brain
Prompts -->	AI ko jo bolte ho	"Mujhe yeh translation karo"
Chains -->	Multiple steps ko connect karna	Pehle translate, phir summarize
Memory -->	Past conversations yaad rakhna	Chatbot ko pehle ki baat yaad
Agents -->	AI ko actions karne dena	Google search, calculator use
Retrievers -->	Documents se relevant info dhunda	PDF se answer nikalna

2. pipe() = ek component ka output → dusre component ka input bana dena

Example: prompt.pipe(model)


3. invoke() = kisi component (prompt / model / chain) ko input dena aur uska output lena

Example: chain.invoke()


4. Output Parser = Model ka response raw text hota hai. Output parser ausse structured format mein convert karta hai (JSON, array, etc.).


5. Chain = Chain ek sequence hai jo multiple LangChain components(prompt / model / outputparser) ko ek pipeline mein connect karti hai. Jaise assembly line mein ek ke baad ek kaam hota hai.

6. Types of Chains:

Single Chain --> Ek component
Sequential Chain --> Multiple steps linear
Router Chain --> Conditional routing

LangChain mein pipe() operator se chains banate hain. Ye functional programming style hai.


7. Memory = Memory woh component hai jo previous conversations store karta hai taake AI context samajh sake.


8. Types of Memory

--> Buffer Memory =	Sab kuch store karta hai	Short conversations
--> Buffer Window Memory =	Recent messages only	Token limit ke liye
--> Summary Memory =	Summary store karta hai	Long conversations
--> Vector Memory =	Semantic search se relevant yaad	Very long conversations

9. PromptTemplate vs ChatPromptTemplate 

👉 PromptTemplate = simple text prompts ke liye
👉 ChatPromptTemplate = chat-based models ke liye (roles ke sath)


10. InMemoryChatMessageHistory = Ye ek storage (memory container) hai jahan chat history store hoti hai.

11. RunnableWithMessageHistory = Ye ek wrapper hai jo tumhari chain + memory ko connect karta hai.

Ye ensure karta hai ke har request ke sath history bhi jaye

12. Document Loaders = PDF, Text file, Webpage se data kaise read karein

13. Text Splitter = Text splitter document ko chhote logical pieces mein todta hai.

14. 🔢 Embeddings: Text ko numbers mein kaise convert karein (taake computer samajh sake)
    💾 Vector Stores: Un numbers ko database mein kaise store karein
    🔍 Similarity Search: Similar text kaise dhundein
